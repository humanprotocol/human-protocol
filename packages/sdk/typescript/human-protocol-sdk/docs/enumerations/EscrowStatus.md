Enum for escrow statuses.

## Enumeration Members

| Enumeration Member | Value | Description |
| ------ | ------ | ------ |
| <a id="launched"></a> `Launched` | `0` | Escrow is launched. |
| <a id="pending"></a> `Pending` | `1` | Escrow is funded, and waiting for the results to be submitted. |
| <a id="partial"></a> `Partial` | `2` | Escrow is partially paid out. |
| <a id="paid"></a> `Paid` | `3` | Escrow is fully paid. |
| <a id="complete"></a> `Complete` | `4` | Escrow is finished. |
| <a id="cancelled"></a> `Cancelled` | `5` | Escrow is cancelled. |
| <a id="tocancel"></a> `ToCancel` | `6` | Escrow is cancelled. |
